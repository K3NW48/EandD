# quick encrypt/decrypt with pgp
    
---------------------------------------------------------------------------------------

Install gpg on Linux or Termux:

apt-get install -y gpg

or

pkg install -y gnupg

---------------------------------------------------------------------------------------

Uses single prefix to quick encrypt/decrypt file after recognizing if .gpg or not.

EXAMPLE: 
  
bash EandD file.txt
    
or
	
bash EandD file.txt.gpg
